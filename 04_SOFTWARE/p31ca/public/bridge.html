BRIDGE

LOVE Economy — Token Dashboard & Phenix Wallet

01. What It Does

The LOVE token is earned by building molecules, completing quests, and minting resonance signatures. Every token earned is engagement evidence — timestamped, exportable, legally admissible. The Bridge dashboard tracks balances, manages vesting schedules, generates stealth addresses, and maintains a full transaction ledger.

02. How It Works

A 4-tab layout (LOVE / WALLET / STEALTH / LEDGER) organizes the full economy interface. SVG circular gauges use stroke-dasharray animation to visualize LOVE balance and spoon energy in real time. Vesting milestones are configured at ages 13, 16, 18, 21, and 25 — each unlock releases a percentage of earned tokens to the child's wallet.

03. Key Features

  • • LOVE balance gauge (0–1000)
  • • Spoon energy tracker (0–12)
  • • Vesting schedules for Bash and Willow
  • • Genesis Gate wallet initialization
  • • Stealth address generation
  • • Transaction ledger with export
  • • Earn / Spend mechanics
