Editorial Suite — Substack Publishing Pipeline

01. What It Does

A complete content creation and publishing system. Organize ideas as seeds, grow them through the draft/review/publish lifecycle, and validate with an 8-point publishing checklist before shipping.

▶ Seed lifecycle (seed → draft → review → published)
▶ Markdown editor with toolbar
▶ 6 framework templates (Posner / Wye-Delta / Tetrahedron / Fuller / SIC-POVM / Spoon Theory)
▶ 8-point publishing checklist
▶ Live word count + reading time
▶ Category tagging
▶ JSON + Markdown export
▶ Auto-save to localStorage

02. How It Works

3-column layout: seed bank on the left, editor in the center, settings on the right. The markdown toolbar inserts formatting at cursor position. A framework library provides quick-insert P31 research concepts for consistent terminology.

Auto-save fires to localStorage on every keystroke, so nothing is lost. The publishing checklist enforces quality gates: title, subtitle, category, tags, featured image, SEO description, proofread status, and final review sign-off.
