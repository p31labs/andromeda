How to read this page

  • Up — probe returned a 2xx response within the timeout window
  • Down — probe failed or timed out
  • Degraded — probe returned an unexpected status that the surface documents as recoverable (e.g. 401 / 403 on auth-required endpoints)
  • Unknown — not yet probed in this session, or CORS-restricted from the browser

Probes are issued from your browser. CORS-restricted endpoints (e.g. some Workers) cannot be probed in-browser; for those, the page shows the URL with the Unknown badge and a link to the operator's glass-box runner. See /glass-box for the canonical operator view.
