Mission Control

Wonky Sprout Token Economy

01. What It Is

Mission Control is the Wonky Sprout token economy for S.J. and W.J. It uses pixel RPG aesthetics (Press Start 2P, neon on dark) to make task completion feel like a game—without dark patterns.

No streaks. No punishments. No time pressure. Missions are completed when they are completed. Tokens accumulate. Rewards unlock.

02. Character Cards

Two character cards — one per kid. Each has a title, an emoji icon, a token balance, and a mission list. Completing a mission awards tokens and triggers a particle burst from the bottom of the screen.

03. Token Economy

GENESIS TOKENS

The currency. Earned by completing missions. Displayed as a running total on each character card. Persist in localStorage between sessions.

MISSION COMPLETION

Tap a mission button. It disables, token balance increments, a toast shows the reward, and particles float up from the bottom. No alert() — zero UI interruption.

SECRET BASE REWARDS

A grid of locked rewards that unlock at token thresholds. Reveal what's behind each tile when the balance crosses the threshold. No spoilers until earned.

NO PUNISHMENTS

Tokens never decrease. Missions never expire. There is no "streak broken" state. The economy only moves forward.
