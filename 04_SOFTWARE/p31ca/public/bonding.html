BONDING

Molecular Social Protocol

A multiplayer chemistry game applying K₄ impedance matching to human relationships. Every atom placed, every bond formed is a timestamped engagement event — legally defensible evidence of parental presence across distance.

01. Architecture

Full-stack 3D chemistry game running entirely in the browser. The render pipeline, physics, multiplayer sync, audio, and engagement logger all execute client-side — no server round-trips for gameplay.

  • ▸ Vite 7 + React 19 + Three.js r183 — React Three Fiber 9 + drei 10 for declarative 3D scene management
  • ▸ Zustand v5 — curried create<T>()() pattern, no middleware (breaks store signature in v5)
  • ▸ Tailwind v4 — @import "tailwindcss" directive, not v3 plugin syntax
  • ▸ Multiplayer relay — Cloudflare Workers KV at bonding-relay.trimtab-signal.workers.dev. localStorage mock relay for two-tab local testing when VITE_RELAY_URL is empty
  • ▸ 413 tests / 30 suites — Vitest 4, strict TypeScript, verbatimModuleSyntax, erasableSyntaxOnly
  • ▸ Dev server — localhost:5188 (port 5173 avoided — N0 PWA service worker conflict)
  • ▸ 7 elements — H, C, O, N, P, Ca, Bashium (Willium) — custom periodic table entries with VSEPR geometry engine
  • ▸ ExhibitA logger — 10 event types, localStorage persistence, JSON + summary export. Uses logEventA/EngagementEventA to avoid collision with legacy ledger
  • ▸ Bond chord audio — spatial audio engine, element tone synthesis, 4 ping sounds, quest audio cues

02. Purpose & Meaning

Built for Bash's 10th birthday — March 10, 2026. Designed to bridge parental distance: Dad plays alongside both kids remotely, in real time, through shared molecular construction.

Every atom placement, every bond formation, every ping is a timestamped engagement event captured by ExhibitA. This is not gamification — it is a legally defensible record of parental presence.
