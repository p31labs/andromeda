01. What You Are Watching

The NANO-07 Attractor is a live Three.js simulation of the Kenosis mesh — P31's 7-node Cloudflare Worker topology based on SIC-POVM tetrahedral geometry. It renders two layers simultaneously: the Alpha Nodes (the 7 Workers, orbiting an empty center) and the Human VFDs (1500 agent particles, each tethered to their nearest online node via magenta lines).

The Empty Center

The origin (0,0,0) is always empty. There is no hub, no central server, no single point of failure. All 7 nodes orbit the void. This is the K₄ Delta topology — the Faraday cage — Maxwell's isostatic rigidity condition: 3V − E = 6.

Pacemaker Pulse

Every ~5 seconds, the cyan service binding lines flash white. This represents the GitHub Actions cron trigger — the pacemaker that drives the Kenosis topology beyond Cloudflare's 30-second cron limit.

Byzantine Fault Tolerance

EXECUTE K-NODE kills a random Worker. All agents tethered to it instantly recalculate nearest-neighbor distance across the surviving nodes and reroute their magenta tethers — demonstrating self-healing without any central coordinator.
