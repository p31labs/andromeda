p31-cortex

AI Agent Orchestration

The limbic system of the P31 mesh. Six autonomous agents — benefits, finance, grant, legal, content, and kofi — run as Cloudflare Durable Objects with D1 SQLite persistence. Cortex provides proactive decision support without requiring operator availability.

01. Architecture

Each agent is a Cloudflare Durable Object — stateful, addressable, persistent. The Orchestrator DO coordinates multi-agent workflows; individual agents own their own state and lifecycle.

BenefitsAgent
Tracks benefit programs, deadlines, eligibility windows
FinanceAgent
Budget tracking, expense categorisation, cashflow alerts
GrantAgent
ARPA-H, ESG, and other funding deadlines + draft assistance
LegalAgent
501(c)(3) compliance requirements, legal deadline surfacing
ContentAgent
Drafts social posts for Kenosis dispatch pipeline
KofiAgent
Processes Ko-fi webhooks, classifies contributions, fires telemetry
  • ▸ BaseAgent class — common lifecycle: wake, process, sleep. Each agent extends BaseAgent and overrides the process method.
  • ▸ D1 SQLite database — 0001_initial.sql schema + 0002_seed_legal_deadlines.sql pre-loads critical compliance dates at deploy time.
  • ▸ Orchestrator DO — coordinates multi-agent workflows, manages agent wake/sleep scheduling, routes inter-agent messages.
  • ▸ Discord notification pipeline — src/notify/discord.ts routes agent outputs to Discord channels via webhook.
  • ▸ Structured telemetry — all agents emit structured telemetry events consumed by Genesis Gate's governance pipeline.

02. Operational Role

Cortex is the "limbic system" of the P31 mesh — the layer that handles autonomic functions so the operator's cognitive bandwidth is reserved for higher-order decisions. It does not require the operator to be present or even aware of routine operations.

GRANT PIPELINE — APRIL 13, 2026

ESG grant window opens April 13, 2026. GrantAgent has this deadline pre-seeded in D1. It will surface preparation requirements, draft summaries, and alert via Discord as the date approaches — without operator prompting.

LegalAgent surfaces 501(c)(3) compliance requirements proactively — annual report deadlines, state registration renewals, and IRS filing windows are pre-loaded from 0002_seed_legal_deadlines.sql.

ContentAgent drafts social content for the Kenosis dispatch pipeline. The content does not post directly — it enters the Kenosis mesh where it is subject to governance hook clearance before dispatch.
