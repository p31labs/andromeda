How the launch button works (3 steps)
  1. On your operator machine, run npm run command-center from the bonding-soup repo root.
  2. Click the launch button above (or open http://127.0.0.1:3131/vibe directly).
  3. Three modes: chat (PHOS personas), cmd (whitelisted scripts), view (read AGENTS.md, CWPs, Manifesto, voice docs inline). After any cmd run, an Ask PHOS for the next move button appears beneath the output.

Source of truth: command-center-terminal.html in bonding-soup. Served from :3131/term, :3131/terminal, and :3131/vibe. New URL in front of an existing service — no new code, no new attack surface. Cross-origin probe from this hub page to your local 127.0.0.1:3131 would be blocked by mixed-content rules; the launch button is the honest path.
