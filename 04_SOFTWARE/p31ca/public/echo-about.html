What Is It

Private loop station: record your voice, loop it, layer up to 4 tracks. Vocal Looper & Script Rehearsal is a core component of the P31 Labs sovereign cognitive infrastructure stack — built to run offline, deployed to the edge, and designed without dark patterns.

MediaRecorder captures the microphone stream. Pitch-shift uses a phase vocoder implemented in Web Audio API ScriptProcessorNode (legacy API — will migrate to AudioWorklet).

Core Features
  • Private loop station: record your voice, loop it, layer up to 4 tracks
  • Real-time pitch-shift: abstracts your voice so you can hear yourself without cringe
  • Situational script practice: write scripts, record read-throughs, loop for rehearsal
  • Offline-first: recordings stay local — nothing uploaded, nothing shared
  • Regulating echolalia mode: loop a calming phrase at variable rate and pitch
How To Use
1Click Record and speak — the first loop captures automatically
2Add pitch shift with the slider — shift up or down to reduce cringe factor
3Switch to Script mode: paste a script, record yourself reading it, replay for practice
