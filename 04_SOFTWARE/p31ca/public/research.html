System Architecture

The P31 mesh is built on a K₄ tetrahedral topology — four vertices, six edges, one center. Every family member is a vertex; every relationship is an edge. The cage structure provides isostatic rigidity: no single point of failure can collapse the whole.

Accessibility & WCAG

Every public page passes WCAG 2.2 AA and targets AAA where feasible. We measure cognitive load, not just color contrast.

  • Skip links and focus indicators on every page
  • Gray Rock mode for sensory overwhelm (no motion, no sound)
  • Reduced motion honored at the browser level
  • Cognitive Load Index scoring per interaction
