Phenix OS

Neurodivergent Operator Command Console

01. What It Is

Phenix OS is an AuDHD operator dashboard — a unified command console built around how a neurodivergent mind actually works. It replaces the chaos of 12 open browser tabs with one coherent interface: navigation, communication, body baseline, and focus, all in a single dark-mode panel.

Named for the mythological bird that rebuilds from collapse. The console assumes the operator will hit walls. It is designed to reduce the cost of re-entry.

02. Five Tabs

  • ▸
    CORE — Daily Baseline Grounding check-in, medication tracker with time-since display, calcium monitor tied to the P31 Ca²⁺ protocol, and a daily reset at midnight.
  • ▸
    MESH — SIC-POVM Navigator Live Three.js tetrahedron representing the 7-node Kenosis mesh. SIC-POVM measurement axes rendered as colored vertex spheres. Rotate to orient. Tap nodes to surface their status.
  • ▸
    FORGE — Chaos Ingestion Inbox for incoming chaos. Paste any message, task dump, or overwhelm trigger. The Forge routes it: discard, defer, delegate, or execute. One decision at a time.
  • ▸
    HEALTH — Body Telemetry Focus timer (Pomodoro-adjacent, no failure state), water intake tracker, and the Fawn Guard comm shield — a pre-send message scorer that flags social compliance patterns before they go out.
  • ▸
    LIBRARY — Encrypted Bottles Private note system. AES-GCM encrypted in localStorage. Each note is a sealed bottle — open with passphrase, store thoughts that should not auto-surface.

03. Design Constraints

NO TIMERS WITH PUNISHMENT

Focus timer counts up, not down. There is no "session failed" state. Walking away and coming back is a supported action.

NO ALERT()

All notifications are inline toast messages. Nothing blocks the UI. The operator decides when to read feedback.

LOCAL-FIRST

All state in localStorage. No account required. No data leaves the device. Encryption at rest for sensitive notes.

DARK DEFAULT

Light mode is the override. Dark is the baseline. Photosensitivity is not an afterthought.
