# P31 TypeScript/React Instructions

- Strict TypeScript. No `any` types.
- Functional components with hooks.
- Tailwind CSS classes for styling.
- Named exports. PascalCase for components, camelCase for hooks/utils.
- All interactive elements need ARIA labels and keyboard event handlers.
- Game actions must emit timestamped log events.
- Web Audio API: create AudioContext only after user gesture.
- Element frequencies: H=523Hz, C=262Hz, O=330Hz, Na=196Hz, P=172Hz, Ca=147Hz.
