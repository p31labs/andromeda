# рҹ”ә P31 ANDROMEDA: SOVEREIGN NODE PORTABLE

Welcome to the Delta Age. This package contains the entire P31 Andromeda mesh ecosystem, designed to run completely offline or tethered to your own cloud infrastructure.

## рҹӣ пёҸ Requirements
* Node.js (v18+)
* Docker & Docker Compose

## рҹҡҖ Installation & Initialization

1. **Unzip** this archive into your desired directory.
2. **Run the AI Setup Wizard** to configure your environment variables:
   ```bash
   node scripts/setup_node.js
   ```
3. **Launch the Mesh** using Docker Compose:
   ```bash
   docker-compose up -d
   ```

## рҹҢҗ Ecosystem Ports

Once initialized, your local node will serve the following interfaces:

| Service | URL | Description |
|---------|-----|-------------|
| FastAPI Backend | http://localhost:8000 | Q-Suite, spoons API, core logic |
| Local Cloudflare Edge | http://localhost:8787 | Workers, KV, social drop automation |
| Discord Bot | (background) | Connects to your Discord server |

## рҹ“Ұ What's Included

- `04_SOFTWARE/` вҖ” Full P31 codebase (backend, workers, firmware, frontend)
- `docker-compose.yml` вҖ” One-command mesh orchestration
- `scripts/setup_node.js` вҖ” Zero-dependency setup wizard
- `.env` вҖ” Generated by wizard (not included in ZIP)

## рҹ”§ Configuration

The setup wizard generates a `.env` file with:

- `NODE_NAME` вҖ” Your sovereign node identifier
- `NODE_ENV` вҖ” production/development
- `DISCORD_TOKEN` вҖ” Bot token (optional)
- `DISCORD_WEBHOOK_URL` вҖ” Webhook for notifications (optional)
- `CLOUDFLARE_API_TOKEN` вҖ” For live edge deployment (optional)
- `CLOUDFLARE_ACCOUNT_ID` вҖ” Cloudflare account (optional)

## рҹҢҚ Deployment Options

### Local (Default)
```bash
docker-compose up -d
```

### VPS / Cloud
```bash
# On your server
scp p31-sovereign-node-*.zip user@server:~/
ssh user@server
unzip p31-sovereign-node-*.zip
cd p31-sovereign-node
node scripts/setup_node.js
docker-compose up -d
```

### Tethered to Live Cloudflare
Run the setup wizard and provide your Cloudflare API token. The local edge worker will proxy to live Cloudflare services.
