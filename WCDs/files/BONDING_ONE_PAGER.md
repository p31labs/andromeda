# BONDING

## Overview

BONDING is an interactive, browser-based chemistry simulation where children build real molecules atom-by-atom in a 3D environment. Developed by P31 Labs as both an educational tool and assistive technology, the platform bridges STEM education and remote family engagement through cooperative play.

## Educational Methodology

The platform uses discovery-based learning aligned with Next Generation Science Standards (NGSS). Molecules are rendered using VSEPR (Valence Shell Electron Pair Repulsion) molecular geometry in an interactive 3D space. Instead of memorizing formulas, children physically connect atoms — building an experiential understanding of chemical bonds, valency, and spatial reasoning through play.

## Assistive Technology

BONDING is designed neurodivergent-first. There is no time pressure, no failure state, and no extraction mechanics. The environment is sensory-friendly and highly predictable, allowing children with varying cognitive and processing needs to explore chemistry at their own pace without experiencing cognitive overload or frustration.

## Remote Parental Engagement

The platform features cooperative multiplayer where a parent and child can build molecules together from separate devices. Every atom placed is logged with a timestamp, creating a transparent and verifiable record of shared activities. This objective engagement data supports families navigating physical separation, remote visitation, or co-parenting arrangements.
