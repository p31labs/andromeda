# P31 Labs Website Research Prompt for Gemini

## Objective
Generate a comprehensive research report to improve the P31 Labs website (phosphorus31.org/planetary-planet) based on current best practices for neurodivergent accessibility, nonprofit transparency, and modern web development.

## Research Areas

### 1. Neurodivergent Accessibility Best Practices (2026)
- **ADHD-Friendly Design**: Current research on attention retention, cognitive load reduction, and task completion optimization
- **Dyslexia Support**: Latest typography research, spacing guidelines, and reading assistance technologies
- **Autism Spectrum**: Sensory-friendly design patterns, predictable navigation, and reduced overwhelm strategies
- **Executive Function**: Tools for task management, memory support, and decision fatigue reduction
- **Universal Design**: WCAG 2.2+ compliance and beyond for neurodivergent users

### 2. Nonprofit Transparency Standards
- **Financial Transparency**: Best practices for real-time financial reporting and donor trust building
- **Impact Measurement**: Modern frameworks for demonstrating social impact to neurodivergent communities
- **Governance**: Transparency requirements for 501(c)(3) organizations serving vulnerable populations
- **Community Engagement**: Strategies for involving neurodivergent voices in organizational decision-making

### 3. Modern Web Development Trends
- **Performance Optimization**: Latest techniques for fast loading and smooth interactions
- **Accessibility Technologies**: Emerging tools and frameworks for inclusive web development
- **Progressive Web Apps**: Benefits and implementation strategies for assistive technology
- **AI Integration**: Ethical AI applications for accessibility and user support

### 4. Assistive Technology Landscape
- **Competitive Analysis**: Leading assistive technology websites and their accessibility features
- **User Experience Patterns**: What works best for neurodivergent families seeking support
- **Content Strategy**: Effective communication methods for complex technical information
- **Community Building**: Online platforms that successfully serve neurodivergent communities

## Specific Questions to Research

1. What are the latest research findings on web design elements that reduce cognitive load for ADHD users?
2. How do current dyslexia-friendly websites implement font choices, spacing, and color contrast?
3. What financial transparency practices build the most trust with neurodivergent communities?
4. What are the most effective ways to present complex technical information to neurodivergent audiences?
5. How can the website better serve as a "Glass Box" while maintaining user privacy and security?
6. What emerging technologies could enhance accessibility without creating dependency on proprietary solutions?
7. How do successful nonprofit websites balance transparency with operational security?
8. What are the latest standards for mobile accessibility for neurodivergent users?

## Deliverables
- Comprehensive research report with actionable recommendations
- Prioritized list of improvements based on impact vs. implementation effort
- Specific technical recommendations for the current Astro/Tailwind implementation
- Examples of best-in-class websites for reference
- Implementation timeline suggestions

## Target Audience
- Neurodivergent individuals (ADHD, Autism, Dyslexia, etc.)
- Parents and caregivers of neurodivergent children
- Healthcare professionals and educators
- Donors and supporters of assistive technology
- Developers and technologists in the accessibility space

## Success Metrics
- Improved accessibility compliance (WCAG 2.2 AA+)
- Enhanced user engagement and task completion rates
- Increased donor trust and transparency scores
- Better search engine visibility for assistive technology keywords
- Positive feedback from neurodivergent community testing
