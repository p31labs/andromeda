# P31 Labs — Claude Rules

Read `context.md` in the project root for full operator and project context.

## Always
- TypeScript strict. No `any`.
- React functional components + hooks.
- Tailwind CSS. No CSS modules.
- Named exports. One component per file.
- Conventional commits.
- ARIA labels on all interactive elements.
- Timestamp every game action for the engagement log.

## Never
- Military, naval, or submarine metaphors (legal sensitivity).
- `localStorage` or `sessionStorage` (deployment constraint).
- `eval()` or dynamic code execution.
- Secrets or API keys in source.
- Class components.
- `@ts-ignore` or `@ts-expect-error` without justification comment.
