## Emergency Protocol

### Burnout Prevention
1. **Recognize:** Irritability, confusion, physical symptoms
2. **Stop:** All non-essential activities immediately
3. **Rest:** Minimum 30 minutes, preferably 2+ hours
4. **Support:** Request Oxygen support via Discord/Ko-fi

### Support Network Activation
- **Primary:** Trusted community members (Oxygen role)
- **Secondary:** Technical support (Carbon role)
- **Emergency:** Medical/professional help if needed
