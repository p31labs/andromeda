## Why This Matters

P31 Labs builds the tools neurodivergent families need — for free, forever. BONDING is a multiplayer chemistry game that doubles as a parental engagement logger (court-admissible timestamped records). The hardware devices (Node Zero, Node One) provide haptic grounding for calcium regulation disorders. The entire stack is open-source, offline-first, and designed for Android tablets.

Every line of code is tested. Every test passes. Every product deploys to the edge. The infrastructure survives without the operator (Kenosis mesh).
